# SEO Audit Checklist for Nimbus Heat Pumps (2025)

This checklist covers essential SEO elements for a UK heating and heat pump website, focusing on 2025 trends, GBP schema implementation, and conversion optimization.

## Technical SEO
- [ ] HTTPS enabled and properly configured
- [ ] Mobile-friendly design (Google Mobile-First Indexing)
- [ ] Page speed optimized (Core Web Vitals: LCP <2.5s, CLS <0.1, FID <100ms)
- [ ] XML sitemap submitted to Google Search Console
- [ ] Robots.txt properly configured
- [ ] Structured data (Schema.org) implemented for GBP and LocalBusiness
- [ ] Canonical URLs set correctly
- [ ] No duplicate content issues
- [ ] 404 errors monitored and fixed
- [ ] Internal linking structure optimized

## On-Page SEO
- [ ] Title tags optimized (50-60 characters, include primary keyword)
- [ ] Meta descriptions compelling (150-160 characters, include CTA)
- [ ] H1 tags present and unique per page
- [ ] Semantic headings (H1-H6) used properly
- [ ] Alt text 100% on all images (descriptive, include keywords where natural)
- [ ] URL structure clean and keyword-rich
- [ ] Keyword density appropriate (1-2% for primary, 0.5-1% for secondary)
- [ ] Content quality high (E-E-A-T: Experience, Expertise, Authoritativeness, Trustworthiness)
- [ ] Local keywords integrated (city names, "near me", postcodes)

## GBP (Google Business Profile) Schema & Local SEO
- [ ] GBP schema markup implemented on contact/service pages
- [ ] LocalBusiness schema with complete business information
- [ ] Review schema for customer testimonials
- [ ] Service schema for heating services offered
- [ ] Organization schema with contact details
- [ ] Local reviews integrated and schema marked up
- [ ] Google My Business profile claimed and optimized
- [ ] Consistent NAP (Name, Address, Phone) across all platforms
- [ ] Local keywords in content and meta tags

## Content & User Experience
- [ ] Content pillars established for heating topics (boilers, heat pumps, energy efficiency)
- [ ] User intent addressed (informational, navigational, transactional)
- [ ] Conversion funnels optimized (quote forms, contact CTAs)
- [ ] Bounce rate monitored and content improved accordingly
- [ ] Internal search functionality working
- [ ] Breadcrumb navigation implemented
- [ ] FAQ schema for common heating questions
- [ ] Video content optimized with transcripts and schema

## Performance & Analytics
- [ ] Google Analytics 4 properly set up
- [ ] Google Search Console configured
- [ ] Core Web Vitals monitored
- [ ] Conversion tracking implemented (quote requests, phone calls)
- [ ] Local pack rankings tracked
- [ ] Voice search optimization (question-based content)
- [ ] AI-generated content reviewed for quality and uniqueness

## UK-Specific Considerations
- [ ] Boiler Upgrade Scheme (BUS) content optimized
- [ ] Energy efficiency ratings prominently displayed
- [ ] Local authority regulations referenced
- [ ] Seasonal content for heating demands
- [ ] Trustpilot/Review platforms integrated
- [ ] Gas Safe registered badges displayed
- [ ] MCS accredited for renewable installations

## 2025 Trends
- [ ] AI-powered chatbots for lead generation
- [ ] Voice search optimization for "best boiler installer near me"
- [ ] Sustainability and green energy focus
- [ ] Video SEO for installation process content
- [ ] Local SEO with hyper-local targeting
- [ ] Zero-click searches addressed with featured snippets
- [ ] Privacy-focused SEO (post-cookie world)

## Monitoring & Maintenance
- [ ] Monthly ranking checks for primary keywords
- [ ] Backlink profile monitored
- [ ] Algorithm update impact assessed
- [ ] Competitor analysis performed quarterly
- [ ] Content calendar for seasonal heating topics
- [ ] Technical issues resolved within 24 hours
